Xrm.Portal.js

A JavaScript library for Power Pages (previously Power Apps portals ). To write basic functions like show and hide in Power Pages sites, we need to rely on jQuery for extensive manipulation of DOM. The library provides these functionality in simple, clear and readable syntaxes. The library will help developers who are familiar with Dataverse client script.

Check out this simple example on setting values:

Xrm.Portal.Form.get(fieldName).setValue(object); Xrm.Portal.Form.get(fieldName).setValue(id, name, entityLogicalName); //For lookup

Check out this simple example on making fields required:

Xrm.Portal.Form.get(fieldName).setRequired(bool); //Default - a default validation and a default message Xrm.Portal.Form.get(fieldName).setRequired(bool, function, message); //A custom validation and a custom message

Other examples:

//Show or Hide Fields Xrm.Portal.Form.get(fieldName).setVisible(bool); //Enable or Disable Xrm.Portal.Form.get(fieldName).setDisable(bool); //Attach Event to field Xrm.Portal.Form.get(fieldName).attachOnChange(callback); //Remove Event from field Xrm.Portal.Form.get(fieldName).removeOnChange(); //Various Datepicker features Xrm.Portal.Form.get(dateTimeFieldName).getData().options({options}); Xrm.Portal.Form.get(dateTimeFieldName).getData().options({ sideBySide: true}); //To show date component and time component side by side Xrm.Portal.Form.get(dateTimeFieldName).getData().minDate(new Date()); //To disable selecting past date
